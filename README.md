# BookManager2-Demo

这是一个基于 BookManager2 项目的演示仓库。

## 项目介绍

BookManager2 是一个图书管理系统，提供了以下功能：
- 图书信息管理
- 图书类型管理
- 借阅管理
- 用户管理

## 使用说明

1. 克隆项目到本地
2. 导入到 IDE (如 IntelliJ IDEA)
3. 配置数据库连接
4. 运行项目
